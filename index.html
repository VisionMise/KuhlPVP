visionPVP

A PVP / PVE Controller

visionPVP allows PVP and PVE to be controlled by time of day, or static modes. visionPVP currently supports 4 modes

  • PVP: PVP is enabled all of the time
  • PVE: PVP is disabled all of the time
  • Night Mode: PVP is only enabled at night (Default Mode)
  • Day Mode: PVP is only enabled during the day
  • Random: PVP and PVE are toggled randomly
Chat Commands
/pvp

Prints the current Server PVP Mode to chat. Available to all players

Console Commands
visionPVP.pvp <setting>

Updates the pvp configuration from the console or RCON
Example: 

visionPVP.pvp pvp-night

visionPVP.rnd

Generates a random number of hours between the min and the max and sets the next pvp/pve toggle time to (the current time) + (the generated number of hours)
Example: 

visionPVP.rnd

visionPVP.rndmin <setting>

Updates the random minimum from the console or RCON. The range is 0 to 24.
Example: 

visionPVP.rndmin 4

visionPVP.rndmax <setting>

Updates the random maximum from the console or RCON. The range is 0 to 24.
Example: 

visionPVP.rndmax 21

Modes
Valid values for config or for console command are as follows:
pvp-day
pvp-night
pvp
pve
random
